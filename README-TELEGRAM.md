# Настройка Telegram бота для отправки заказов

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Simple Trading Orders")
   - Введите username бота (должен заканчиваться на `bot`, например: `simpletrading_orders_bot`)
4. BotFather отправит вам **токен бота** (выглядит примерно так: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. **Сохраните этот токен!**

## Шаг 2: Получение Chat ID

### Вариант А: Используя бота @userinfobot
1. Найдите в Telegram **@userinfobot**
2. Отправьте ему `/start`
3. Бот покажет ваш **Chat ID** (число, например: `123456789`)

### Вариант Б: Через ваш бот
1. Найдите созданного бота и отправьте ему любое сообщение (например: "Привет")
2. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Найдите в ответе поле `"chat":{"id":123456789}` - это ваш Chat ID

## Шаг 3: Настройка на Vercel

1. Зайдите на https://vercel.com
2. Откройте ваш проект
3. Перейдите в **Settings → Environment Variables**
4. Добавьте две переменные:
   - **Name:** `TELEGRAM_BOT_TOKEN`, **Value:** токен от BotFather
   - **Name:** `TELEGRAM_CHAT_ID`, **Value:** ваш Chat ID
5. Сохраните изменения
6. Сделайте **Redeploy** проекта (Deployments → последний деплой → три точки → Redeploy)

## Шаг 4: Проверка

1. После деплоя откройте ваш сайт
2. Заполните форму заказа
3. Нажмите "Отправить заказ"
4. Проверьте Telegram - вам должно прийти сообщение от бота

## Возможные проблемы

### "Сервер не настроен"
- Убедитесь, что добавили обе переменные окружения на Vercel
- Проверьте правильность написания имён переменных (регистр важен!)
- Сделайте Redeploy после добавления переменных

### "Ошибка отправки в Telegram"
- Проверьте правильность токена бота
- Убедитесь, что отправили хотя бы одно сообщение боту
- Проверьте Chat ID

### Бот не отвечает
- Убедитесь, что начали диалог с ботом (отправили /start)
- Проверьте, что бот не заблокирован

## Безопасность

⚠️ **Важно:**
- Никогда не публикуйте токен бота в публичных репозиториях
- Файл `.env` добавлен в `.gitignore` для защиты токенов
- Используйте Environment Variables в Vercel, а не файл `.env` в продакшене

## Дополнительные настройки бота (опционально)

Вы можете настроить бота через @BotFather:
- `/setdescription` - описание бота
- `/setabouttext` - текст "О боте"
- `/setuserpic` - аватар бота
