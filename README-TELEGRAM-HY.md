# Telegram բոտի կարգավորում պատվերների համար

## Քայլ 1: Telegram բոտի ստեղծում

1. Բացեք Telegram-ը և գտեք **@BotFather**
2. Ուղարկեք `/newbot` հրամանը
3. Հետևեք հրահանգներին:
   - Մուտքագրեք բոտի անունը (օրինակ՝ "Simple Trading Orders")
   - Մուտքագրեք բոտի օգտանունը (պետք է ավարտվի `bot`-ով, օրինակ՝ `simpletrading_orders_bot`)
4. BotFather-ը կուղարկի **բոտի տոկենը** (նման է սրան՝ `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. **Պահպանեք այս տոկենը!**

## Քայլ 2: Chat ID-ի ստացում

### Տարբերակ Ա: @userinfobot-ի միջոցով
1. Telegram-ում գտեք **@userinfobot**
2. Ուղարկեք `/start`
3. Բոտը կկարծավորի ձեր **Chat ID**-ն (թիվ, օրինակ՝ `123456789`)

### Տարբերակ Բ: Ձեր բոտի միջոցով
1. Գտեք ձեր բոտը և ուղարկեք նրան ցանկացած հաղորդագրություն
2. Բրաուզերում բացեք՝ `https://api.telegram.org/bot<ՁԵՐ_ՏՈԿԵՆ>/getUpdates`
3. Պատասխանում գտեք `"chat":{"id":123456789}` դաշտը - սա ձեր Chat ID-ն է

## Քայլ 3: Vercel-ում կարգավորում

1. Մուտք գործեք https://vercel.com
2. Բացեք ձեր նախագիծը
3. Անցեք **Settings → Environment Variables**
4. Ավելացրեք երկու փոփոխական:
   - **Name:** `TELEGRAM_BOT_TOKEN`, **Value:** BotFather-ից ստացված տոկեն
   - **Name:** `TELEGRAM_CHAT_ID`, **Value:** ձեր Chat ID
5. Պահպանեք փոփոխությունները
6. Կատարեք **Redeploy** (Deployments → վերջին deploy → երեք կետ → Redeploy)

## Քայլ 4: Ստուգում

1. Deploy-ից հետո բացեք ձեր կայքը
2. Լրացրեք պատվերի ձևը
3. Սեղմեք "Ուղարկել"
4. Ստուգեք Telegram-ը - բոտից հաղորդագրություն պետք է ստանաք

## Հնարավոր խնդիրներ

### "Սերվերը կարգավորված չէ"
- Համոզվեք, որ Vercel-ում ավելացրել եք երկու փոփոխականներ
- Ստուգեք փոփոխականների անունների ճիշտությունը
- Կատարեք Redeploy փոփոխականներ ավելացնելուց հետո

### "Telegram-ին ուղարկելու սխալ"
- Ստուգեք տոկենի ճիշտությունը
- Համոզվեք, որ բոտին առնվազն մեկ հաղորդագրություն եք ուղարկել
- Ստուգեք Chat ID-ն
